# Ajustar aviso de ayuda de stats

**Features**
- [x] Mostrar el aviso cuando se agrega un stat al canvas, en lugar de al primer toque.
- [x] Mantener el aviso como ayuda breve para descubrir colores y estilo.
- [x] Cambiar el texto a: `tap stat for colors & style`.

**Design**
- [x] Hacer el aviso aproximadamente a la mitad del tamaño actual para que se sienta más discreto.
- [x] Colocarlo mucho más cerca del borde superior del canvas.
- [x] Mantener una apariencia visible pero elegante, sin tapar el contenido principal.

**Pages / Screens**
- [x] Editor de foto: el aviso aparecerá justo después de agregar un stat al canvas y seguirá funcionando como pista visual breve.
