# Igualar el estilo del stat en el canvas

**Features**

- [x] El stat que aparece en el canvas tendrá el mismo estilo visual que el stats card que ya quedó bien.
- [x] Los números usarán la misma presencia, proporción y sensación de ancho que en la versión corregida.
- [x] La etiqueta pequeña del stat también mantendrá el mismo tratamiento visual para que todo se vea uniforme.
- [x] La vista final y la exportación conservarán ese mismo look, sin diferencias visuales entre tarjeta y canvas.

**Design**

- [x] Mantener la misma jerarquía tipográfica ya aprobada en el stats card.
- [x] Hacer que el número destaque con el mismo peso visual y balance horizontal.
- [x] Ajustar el espaciado para que el conjunto se vea limpio, parejo y consistente dentro del canvas.
- [x] No cambiar el estilo general del editor, solo corregir esta inconsistencia visual.

**Pages / Screens**

- [x] Editor de foto: el stat dentro del canvas se verá igual que su versión correcta.
- [x] Resultado exportado: conservará el mismo estilo visual del canvas corregido.
