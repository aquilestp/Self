# Settings screen adjustments


## Changes

- [x] **Bajar título "Settings"**: Se añade un pequeño desplazamiento hacia abajo al título en la barra de navegación.

- [x] **Push Notifications desactivadas**: Cuando las notificaciones estén desactivadas o no permitidas, el card completo se convierte en un botón con un subtexto tipo "Tap to enable in Settings" y al hacer click abre directamente los ajustes del sistema de iOS para que el usuario las active.

- [x] **"Delete Account" color**: El texto "Delete Account" cambia a blanco con 50% de transparencia (`Color.white.opacity(0.50)`), al igual que su ícono de basura.
